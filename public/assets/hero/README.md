# Hero Carousel Images

This folder contains the images for the hero carousel section.

## Required Images

You need 3 images for the carousel slides:

1. **slide1.jpg** - Premium wine bottles and glass (Martell style)
   - Recommended size: 1920x1080px
   - Theme: Luxury spirits, dark background, gold accents

2. **slide2.jpg** - Vintage wine cellar or Bordeaux bottles
   - Recommended size: 1920x1080px
   - Theme: Wine barrels, aged bottles, warm lighting

3. **slide3.jpg** - Champagne or celebration scene
   - Recommended size: 1920x1080px
   - Theme: Champagne bottles, flutes, ice bucket

## How to Add Your Images

1. Download or prepare your 3 wine images
2. Rename them to: `slide1.jpg`, `slide2.jpg`, `slide3.jpg`
3. Place them in this folder: `public/assets/hero/`
4. The carousel will automatically use them

## Image Sources (Suggestions)

- **Unsplash**: https://unsplash.com/s/photos/wine
- **Pexels**: https://www.pexels.com/search/wine/
- **Pixabay**: https://pixabay.com/images/search/wine/

Search for: "premium wine", "wine bottle dark", "martell", "wine cellar", "champagne luxury"
